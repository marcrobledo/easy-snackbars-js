# Easy Snackbars JS

Lightweight library to easily add customizable [snackbars](https://m2.material.io/components/snackbars) to your website and webapps.

![Easy Snackbars JS screenshot](example.jpg?raw=true)

See [example demo, installation instructions and documentation](https://www.marcrobledo.com/easy-snackbars-js/)

Features:
- Lightweight
- Easy to install and configure
- Customizable
- Made with vanilla JS
